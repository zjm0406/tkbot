## 核心内容
遗忘系统的衰减曲线不应该是均匀的——它应该有"潮汐周期"。某些时间段某些问题更可能被激活。seed-extractor 可以加入"认知水位"参数：低水位暴露常规种子，高水位暴露深层种子。

## 为什么暂存
这个想法很有潜力但目前缺少具体实现方案——"潮汐周期"的具体参数（周期多长？基于什么信号判断水位？）还需要进一步设计。
