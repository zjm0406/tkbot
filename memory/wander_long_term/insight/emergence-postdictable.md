## 核心结论
涌现的核心悖论——"事后可解释但事前不可预测"——精准定义了 Wander Mode 的目标：寻找那些回头看觉得合理、但事前想不到的联结。

## 推导路径
LLM 涌现能力 → 复杂系统自组织 → 经济市场"看不见的手" → 意识涌现 "hard problem" → 确定性随机

## 对 Wander Mode 的指导意义
- 评估基准：好的联想 = 事后必然感 + 事前意外感 的组合
- 新颖性（Novelty）评分维度可据此细化
- 如果联想结果"事前完全可预测"，说明发散不够
- 如果联想结果"事后仍然不合理"，说明连贯性（Coherence）不足

## 后续行动
将此原则纳入 value-filter 的 Novelty 和 Coherence 评分指南
