## 核心结论
任何生成性系统都需要在无序扩散（熵增）与有序筛选（信息压缩）之间建立动态平衡——这种平衡本身就是一种认知涌现。

## 推导路径
Wander Mode 的发散-收敛双过程 → 热力学熵增类比 → 信息论编码压缩 → 认知涌现作为平衡态

## 对 Wander Mode 的指导意义
- 发散阶段 = 增大可能性空间（positive entropy）
- 收敛阶段 = 信息压缩（negative entropy / negentropy）
- 评分阈值（18/14）本质上是压缩率参数
- 最佳停止点 = 边际信息增益趋近于零的时刻
