## 核心结论
**遗忘不是记忆系统的缺陷，而是记忆系统的核心功能。** AI 记忆系统的设计应该从"怎么存更多"翻转为"怎么让遗忘决策本身成为最智能的部分"。

## 推导路径
人类记忆巩固 → 睡眠中的突触修剪 → 细胞自噬（autophagy：细胞主动降解自身组分来生存）→ 遗忘管理器不应该只是 Wander Mode 的辅助模块，应该是与 seed-extractor 平级的核心入口。每一轮在提取种子之前，应该先问"什么不值得被记住"。

## 反常识版本
假设"遗忘是敌人"→ 反转"遗忘是核心功能"→ 遗忘模块应该是认知架构的一等公民。

## 可在什么场景复用
- 任何需要长期记忆的 AI 系统
- agent 的上下文管理策略
- 知识库的自动清理和维护
- 认知架构中"积极遗忘"vs"被动遗忘"的设计选择
